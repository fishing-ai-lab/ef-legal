이지피싱 계정 삭제 안내

최종 업데이트: 2026년 3월 7일

이지피싱 앱에서 계정을 삭제(회원탈퇴)하는 방법과 삭제 시 처리되는 데이터에 대해 안내합니다.

1. 계정 삭제 방법

앱 내에서 직접 계정을 삭제할 수 있습니다.

  1. 이지피싱 앱을 실행합니다.
  2. 하단 탭에서 MY 탭을 선택합니다.
  3. 메뉴 목록에서 회원탈퇴를 선택합니다.
  4. 콘텐츠 처리 방식을 선택합니다:
    • 모든 콘텐츠 삭제 — 리뷰, 조과, 댓글 등 모든 데이터 삭제
    • 콘텐츠 유지 — 개인정보만 파기하고 기존 콘텐츠는 유지
  5. 탈퇴하기 버튼을 눌러 탈퇴를 완료합니다.

2. 삭제되는 데이터

데이터 유형 모든 콘텐츠 삭제 콘텐츠 유지
이름, 이메일, 프로필 사진 즉시 삭제 즉시 삭제
조과 기록 (낚시 기록) 삭제 유지 (닉네임으로 표시)
리뷰, 댓글 삭제 유지 (닉네임으로 표시)
찜한 포인트, 내 포인트 삭제 삭제
로그인 정보 즉시 차단 즉시 차단

3. 복구 기간

30일 이내에 동일한 소셜 계정으로 다시 로그인하면 계정을 복구할 수 있습니다.

30일 경과 후에는 계정이 영구 삭제되며 복구할 수 없습니다.

  • "모든 콘텐츠 삭제" 선택 시: 모든 데이터가 완전히 파기됩니다.
  • "콘텐츠 유지" 선택 시: 개인정보만 파기되고, 콘텐츠는 익명으로 유지됩니다.
